## Welcome to the beluga stack! 
In this storybook various design elements and UI components are shown and described. You can change, customize or add anything. 

## Who is this tech stack for?
This tech stack is suitable for:

- Developers who want to build modern web applications with a focus on scalability and maintainability
- Teams looking for a comprehensive, pre-configured development environment
- Projects that require a robust UI component library with customizable design elements
- Startups and enterprises aiming to accelerate their development process with best practices baked in
- Developers interested in leveraging the power of React, TypeScript, and Tailwind CSS in a cohesive ecosystem
- Those who appreciate well-documented components and a streamlined development workflow
- Teams that value component-driven development and want to utilize Storybook for better collaboration between designers and developers

Whether you're building a small prototype or a large-scale application, the beluga stack provides the flexibility and tools to support your project's growth and evolution.

## What does this tech stack contain?
Here are the most important frameworks and technologies used in this project:

1. **Turborepo**: A build tool for monorepos
2. **React**: A JavaScript library for building user interfaces
3. **Next.js**: A React framework for server-side rendered applications
4. **TypeScript**: A typed superset of JavaScript
5. **Tailwind CSS**: A utility-first CSS framework
6. **Storybook**: A tool for developing and documenting UI components
7. **React Query**: A library for data management in React applications
8. **Prisma**: A modern ORM (Object-Relational Mapping) for Node.js and TypeScript
9. **PostgreSQL**: A powerful, open-source database system (with Docker)
10. **Lucia Auth**: An authentication framework for modern web applications

In addition, the following important tools are used:

- **next-safe-action**: A library for type-safe server actions in Next.js
- **Vite**: A modern frontend development server and bundler
- **ESLint**: A tool for static code analysis
- **Prettier**: A code formatter

These technologies form the backbone of the beluga stack and enable efficient, scalable and maintainable development of modern web applications with robust database connectivity, secure authentication and optimized data management.
